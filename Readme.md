# Event Management System (Code.Init() 2024 Hackathon)

This is a fork of the Event Management System developed during **Code.Init() 2024 Hackathon**. The project was a team effort to create a web application for managing and registering events.

## My Contributions
- Developed **responsive frontend** using **HTML, CSS, and JavaScript**.
- Integrated frontend with backend **APIs** for event registration and management.
- Collaborated in a **4-member team** to deliver a working prototype in 48 hours.
- Focused on **UI/UX**, layout, and interactivity of the frontend components.

## Features
- Event creation and registration
- Dynamic event listing
- User-friendly and responsive interface

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: Git, GitHub

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/YourUsername/event-management.git
   ```
2. Follow instructions in README.md in the backend folder to start the server.

3. Open the application in a browser to interact with it.
